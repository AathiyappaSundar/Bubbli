# Bubbli Digital Services

One-Stop Digital Solution for Local Businesses. Everything your local business needs to be found online. Websites, Google Maps, WhatsApp & more — simple, affordable, and done-for-you.

## Features

- **Websites**: Custom, professional websites.
- **Google Maps**: Setup and optimization.
- **WhatsApp Integration**: Easy communication with customers.
- **Affordable**: Starting at ₹2,000.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1.  Clone the repository.
2.  Install dependencies:
    ```sh
    npm install
    ```
3.  Start the development server:
    ```sh
    npm run dev
    ```

## Technology Stack

- **Frontend**: React, Vite, Tailwind CSS, shadcn/ui
- **Language**: TypeScript

## Project Structure

- `src/components`: Reusable UI components and sections
- `src/pages`: Application pages
- `src/lib`: Utilities
